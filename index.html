The Big Picture

Meet Your New Digital Companion

OpenClaw is a self-hosted AI gateway — a command layer that bridges you, your devices, and the most powerful language models in the world. Think of it as your private Jarvis: always on, always learning, always yours.

💬

Omnichannel Intelligence

Chat through WhatsApp, Telegram, Discord, Slack, Signal, iMessage, and more — all responding from one unified AI brain.

🔒

Privacy by Default

Your data never touches an external server. OpenClaw runs locally — on your machine or your cloud VM — giving you full sovereignty over your conversations.

🧠

Model Agnostic

Use Claude Opus 4.5 for deep reasoning, Claude Sonnet 4.5 for everyday tasks, GPT-4 for versatility — or mix models based on context and cost.

⚡

Automation Ready

OpenClaw can browse the web, manage files, run scripts, and orchestrate workflows — all triggered by a simple message on your phone.

🔧

Extensible with Skills

🌐

Always Available

The gateway daemon keeps OpenClaw running 24/7, even when you close your terminal. No babysitting required.

Before You Begin

Prerequisites

OpenClaw is designed to get out of your way. Here's what you need before the first command.

01

A Computer — or a Cloud VM

OpenClaw runs on macOS, Windows, and Linux. For best results and 24/7 availability, consider running it on a dedicated VM — such as Oracle Cloud's Free Tier, which works seamlessly with OpenClaw.

macOS Windows Linux Cloud VM
02

Node.js v22 or Higher

OpenClaw is built on JavaScript, so Node.js is the engine powering it. Verify your version in a terminal:

Terminal
node --version
# Should return v22.x.x or higher

If you need to install or upgrade, download the LTS release from nodejs.org. Alternatively, install via npm directly:

npm (alternative)
npm i -g openclaw
03

An AI Provider API Key

OpenClaw is the intelligence layer — you bring the brain. Choose one (or both):

Recommended
Anthropic Claude
Best with: claude-opus-4.5

Claude excels at long-form reasoning, nuanced instruction-following, and maintaining context. Claude Opus 4.5 is the top recommendation for power users.

Get API Key →
OpenAI GPT
Best with: gpt-4

A widely-supported option with broad ecosystem compatibility. A solid choice if you already have an existing OpenAI subscription.

Get API Key →
⚠️

Keep your API key private. Never share it, commit it to public code repositories, or send it over chat. Treat it like a password.

04

About 30 Minutes

The setup is methodical, not difficult. Follow each step in sequence and you'll have a fully operational AI assistant before your next cup of coffee.

Part 1

Installation

One command to rule them all. The OpenClaw installer handles dependency resolution, path configuration, and initial bootstrapping automatically.

bash 
curl -fsSL https://openclaw.ai/install.sh | bash
ℹ️

curl -fsSL downloads the installer script. Piping it to bash executes it immediately. The installer fetches the latest stable CLI, installs it globally, and prepares your environment.

PowerShell (Run as Administrator) 
iwr -useb https://openclaw.ai/install.ps1 | iex
Terminal / PowerShell 
npm i -g openclaw
ℹ️

Recommended if you already have Node.js v22+ installed. Works identically across all operating systems.

⏳

What to Expect

The installer will download OpenClaw, resolve all dependencies, and configure your system PATH. The process takes 2–5 minutes depending on connection speed. Expect scrolling output — that's intentional.

✅

Verify the Installation

Terminal
openclaw --version
# e.g. v2025.2.14

If the version number appears, you're ready to proceed. If you see "command not found", close and reopen your terminal, then try again.

Part 2

Onboarding Wizard

With OpenClaw installed, a single command launches the interactive setup wizard — walking you through every configuration decision from AI provider selection to channel integration.

Terminal
openclaw onboard --install-daemon
ℹ️

--install-daemon configures OpenClaw to run as a background service. This means it stays active across terminal sessions and system restarts — no manual launching required.

1

Name Your Assistant

Choose anything — Jarvis, Alfred, or something distinctly your own. This name personalizes your assistant's identity and how it refers to itself.

2

Choose Your AI Provider

Select between Anthropic or OpenAI using the arrow keys. Your API key will be securely stored in the local credentials store.

3

Enter Your API Key

Paste your key when prompted. The terminal will not display it as you type — that's a standard security behavior, not an error.

Anthropic console.anthropic.com → API Keys → Create Key
OpenAI platform.openai.com/api-keys → Create new secret key
4

Select a Model

The wizard will display available models. For depth and reasoning, choose claude-opus-4.5. For speed and economy, claude-sonnet-4.5 is an excellent alternative.

ModelStrengthCost
claude-opus-4.5Deep reasoningHigher
claude-sonnet-4.5BalancedMedium
gpt-4VersatileMedium
5

Configure Messaging Channels

Choose which channels you'd like to enable. Use Space to toggle and Enter to confirm. Channels can also be added later via openclaw config edit.

6

Install the Gateway Daemon

Accept when prompted. The daemon registers OpenClaw as a system service, ensuring it restarts with your machine and remains accessible 24/7.

Architecture

The Gateway

The gateway is the central nervous system of OpenClaw — the persistent process coordinating all AI requests, channel routing, and tool execution. It runs on port 18789 and exposes both a REST API and a web dashboard.

Check Status

Terminal
openclaw gateway status

Start Gateway

Terminal
openclaw gateway start

Open Dashboard

Terminal
openclaw dashboard

Restart Gateway

Terminal
openclaw gateway restart
🎯

The web dashboard is accessible at http://127.0.0.1:18789/ — a clean, professional interface where you can chat directly with your assistant, inspect logs, and manage settings.

Part 3

Messaging Channels

The real power of OpenClaw is meeting you wherever you already communicate. Here's how to connect each major platform.

WhatsApp

Medium difficulty

WhatsApp uses QR code authentication. You'll link OpenClaw to a dedicated WhatsApp number — a separate number from your personal one is recommended.

Step 1 — Initiate login
openclaw channels login whatsapp

A QR code will render in your terminal. On your phone: open WhatsApp → three-dot menu → Linked Devices → Link a Device → scan the code.

Step 2 — Approve a contact number
openclaw pairing approve whatsapp +15555551234
ℹ️

By default, OpenClaw only responds to approved numbers. This prevents unauthorized access to your assistant via WhatsApp.

Telegram

Easy — Recommended first channel

Telegram is the most straightforward integration. You'll create an official bot via Telegram's BotFather, then wire the token into OpenClaw.

  1. Open Telegram and search for @BotFather
  2. Send /newbot and follow the naming prompts
  3. Copy the API token BotFather provides
openclaw.json — channels section
{
  "channels": {
    "telegram": {
      "botToken": "your-bot-token-here"
    }
  }
}
Apply changes
openclaw gateway restart

Discord

Moderate — Requires developer portal access
  1. Visit discord.com/developers/applications → New Application
  2. Go to Bot → Add Bot → Reset Token and copy the token
  3. Enable: Presence Intent, Server Members Intent, Message Content Intent
  4. Go to OAuth2 → URL Generator, select bot, enable Send/Read Messages permissions, copy the URL to invite your bot to a server
openclaw.json — channels section
{
  "channels": {
    "discord": {
      "token": "your-bot-token-here"
    }
  }
}

Troubleshooting

Common Issues

Command not found after install

The shell's PATH hasn't refreshed. Close your terminal, open a new session, and retry. If the issue persists, re-run the installer to ensure OpenClaw is globally linked.

Gateway won't start — port already in use
Use alternate port
openclaw gateway --port 18790
AI is not responding to messages

Verify your API key is valid and your provider account has remaining credits. Check the logs for detailed error output:

Terminal
openclaw logs
WhatsApp keeps disconnecting

WhatsApp's web bridge is sensitive to network interruptions. Ensure your host machine (or VM) maintains a stable internet connection. Re-scanning the QR code via openclaw channels login whatsapp will re-authenticate the session.
